# Example queries

Drop one `.md` file per example company-research query in this folder. Freeform — whatever you'd
actually type or paste. Useful things to include when you have them:

- the ask itself ("find me companies that ...")
- hard requirements vs. nice-to-haves, if you'd split them
- the columns you'd want in the output (funding, traffic, headcount, pricing, whatever)
- an existing list to dedupe against (or a note that one exists and its format)
- roughly how many companies you want back

These get turned into worked examples in `references/` and used as live test queries for the skill.
